**Setup Guide**

**Prerequisites**
Python 3.x
pip
Flask

**Install Python**
Download and install Python from: https://python.org

**Install Flask**
Run:
pip install flask

**Run the Script**
Navigate to the folder in the system where the file named "upload_server.py" is downloaded and run the below command:
python upload_server.py

Once the command s executed, it will provide a local IP-based URL.
Open the URL from another system, connected to the same network, you should be able to see the webpage with file upload functonality.
