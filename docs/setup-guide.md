Download the upload_server.py script. After downloading, create a new directory on your system and move the script into it. Within the same directory, create a folder named uploads.
**Note:** Ensure the folder name is exactly uploads, as the script is configured to store and reflect all uploaded content in this directory.

Navigate to the folder in the system where the file named "upload_server.py" is downloaded and run the below command: 
python upload_server.py
